Solution

In my solution I create an array, iterate over all numbers from 1 to 100,
fill the array depend on number.
There can be different situations: 
- 3 and 5 present in a number: 35 and 53. Push FizzBuzz or BuzzFizz.
- 3 or 5 present in a number. Push Fizz or Buzz 
- other digits in a number. Push the number.
        
After that, just use a helper function to print all these values
and place them on a table for better organization.
        
In my test function, I compare an array returned by fizzBuzz() function
with the predefined one. In case they are equals the test passes otherwise
return the reason of the failure.
